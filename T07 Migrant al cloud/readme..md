
# Migrant al Cloud: Solució de Correu Corporatiu per FoodLogistic

## Breu descripció

Estudi de mercat i proposta integral de solucions de correu electrònic corporatiu per a l'empresa FoodLogistic, buscant modernitzar la seva infraestructura de comunicacions i col·laboració online.

## Introducció al cas

FoodLogistic, amb una plantilla de 35 treballadors, necessita actualitzar el seu servei de correu externalitzat. El sistema actual presenta:
- Problemes de seguretat
- Limitacions funcionals (només correu)
- Obsolescència tecnològica

**Objectiu:** Investigar i avaluar solucions de correu corporatiu al mercat per elaborar una proposta comercial i tècnica que equilibri funcionalitats, seguretat i pressupost.
